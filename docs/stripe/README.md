# Stripe & Subscriptions Documentation

This directory contains all Stripe payment and subscription-related documentation.

## Files

- **STRIPE_SETUP_GUIDE.md** - Complete Stripe integration setup
- **STRIPE_QUICK_START.md** - Quick start guide for Stripe
- **STRIPE_TESTING_GUIDE.md** - Testing payments and subscriptions
- **SUBSCRIPTION_LIMITS_STRATEGY.md** - Feature limits for free/pro plans
- **SUBSCRIPTION_QUICK_START.md** - Quick subscription setup
- **SUBSCRIPTION_SYSTEM_DOCUMENTATION.md** - Complete subscription system docs

## Quick Start

1. **STRIPE_QUICK_START.md** - Get Stripe working quickly
2. **STRIPE_SETUP_GUIDE.md** - Complete setup instructions
3. **STRIPE_TESTING_GUIDE.md** - Test payments locally

## Key Concepts

- **Stripe Products**: Define subscription tiers (Free, Pro)
- **Price IDs**: Environment variables for monthly/yearly plans
- **Webhooks**: Handle subscription events (payment succeeded, canceled, etc.)
- **Customer Portal**: Users manage subscriptions themselves
- **Feature Gating**: Limit features based on subscription tier
