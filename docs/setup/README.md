# Setup & Configuration Documentation

This directory contains setup instructions and configuration guides.

## Files

- **SETUP.md** - Initial project setup instructions
- **ENVIRONMENT_VARIABLES.md** - Required environment variables
- **POSTHOG_SETUP.md** - PostHog analytics setup

## Quick Start

1. **SETUP.md** - Follow initial setup instructions
2. **ENVIRONMENT_VARIABLES.md** - Configure all required environment variables
3. **SQL_FILES_CHECKLIST.md** - Run database migrations
